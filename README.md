Task 2: Telecom Customer Churn Analysis Dashboard
📌 Project Overview

The Telecom Customer Churn Analysis Dashboard was developed to analyze customer churn behavior in the telecom industry. The dashboard helps identify key factors that influence customer retention and customer loss, enabling businesses to improve customer satisfaction and reduce churn rates.

🎯 Objectives
Analyze customer churn trends
Identify factors affecting customer retention
Compare churn rates across different customer categories
Generate business insights using visual analytics
📈 Key Insights
Customers with Month-to-Month contracts showed the highest churn rate.
Fiber Optic users had a higher probability of churning.
Customers using Electronic Check payment methods had maximum churn.
Senior Citizens showed a higher churn percentage compared to non-senior citizens.
Customers with longer tenure were more likely to stay with the company.
